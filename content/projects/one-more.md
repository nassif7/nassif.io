oneMore is a cigarette tracking app. You log each one as you smoke it. That's basically it.

The idea is simple: most smokers don't have a clear picture of how much they actually smoke. Seeing the number in real time, noticing the patterns, knowing what triggers it. That alone changes something. No lectures, no streak counters, no pressure.

If you want to cut down, great. If you're not ready, it still shows you where you are. Built for iOS and Android with React Native and Expo.
